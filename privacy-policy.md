# Privacy Policy – Hantavirus Map

Last updated: May 2026

Hantavirus Map is an informational mobile application that displays public outbreak-related events on a map using publicly available sources.

## Information we collect

The app does not require account creation and does not ask users to provide personal information.

The app loads public event data from a remote JSON file hosted on GitHub. This data contains event information such as location, date, case count, source name, and source URL.

## Advertising

This app uses Google AdMob to display ads. Google AdMob may collect device identifiers, advertising identifiers, approximate location, app interactions, diagnostics, and other data according to Google’s advertising and privacy policies.

Users can manage ad personalization through their Google account or device settings.

## Third-party services

The app may use the following third-party services:

- Google AdMob for advertising
- OpenStreetMap / CARTO / Leaflet map tiles for map display
- GitHub for hosting public event data
- External websites when users open cited sources

These services may process data according to their own privacy policies.

## Medical disclaimer

Hantavirus Map is for informational purposes only. It does not provide medical advice, diagnosis, or treatment. Users should always follow official public health authorities and medical professionals.

## Children

The app is intended for users aged 18 and over.
